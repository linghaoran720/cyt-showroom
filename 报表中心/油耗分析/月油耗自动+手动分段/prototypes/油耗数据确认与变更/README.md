# 油耗数据确认与变更控制 原型

## 页面列表
- `index.html` - 月报表主页（含确认/撤回/变更详情等全部弹窗交互）
- `change-notify.html` - 变更提醒列表页

## 使用说明
1. 用浏览器打开 `index.html`
2. 点击页面上的按钮体验交互流程

## 交互说明

### index.html 月报表主页
- **确认本月数据**：点击操作栏"确认本月数据"按钮 → 弹出确认弹窗，展示本月数据概况
- **撤回确认**：点击"撤回确认"按钮 → 弹出撤回弹窗，提示待处理变更将自动更新
- **查看变更**：表格第6行（京F99001，状态为"有变更"）→ 点击"查看变更" → 弹出变更详情弹窗，展示变更前后数据对比
- **铃铛图标**：右上角铃铛带红色角标(3) → 点击跳转到变更提醒列表页
- **确认状态列**：已确认(绿色)、未确认(蓝色)、有变更(橙色) 三种状态

### change-notify.html 变更提醒列表
- 展示所有变更提醒，区分"待处理"和"已处理"
- 第2行（京G22334）为影响满到满周期的变更，高亮显示"影响2个月"标记
- 点击"确认更新"可直接处理变更
- 点击"查看详情"跳转回月报表页面

### 弹窗类型
1. **确认本月弹窗**：按月整体确认，展示车辆统计
2. **撤回确认弹窗**：提示撤回影响，待处理变更将自动执行
3. **普通变更弹窗**：不影响周期的变更，展示数据对比（变更前/后/差异）
4. **影响周期变更弹窗**：加满记录变更影响满到满周期，红色醒目提示涉及2个月份数据变化
