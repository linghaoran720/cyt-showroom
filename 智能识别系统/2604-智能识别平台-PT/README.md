# 智能识别平台 原型

## 页面列表
- index.html - 单个运单图片识别（左右分屏）
- batch-table.html - 批量识别表格模式
- mapping-management.html - 映射规则管理

## 使用说明
1. 直接用浏览器打开 index.html
2. 点击页面上的链接可跳转到其他页面

## 交互说明
- **单个识别**：上传图片 → 识别 → 左右分屏展示 → 编辑映射 → 创建运单
- **批量识别**：上传多个图片/文字/Excel → 双表头表格展示 → 编辑内容 → 批量创建
- **映射管理**：查看映射规则 → 搜索 → 删除

- 电脑端：desktop-design-standard.md
