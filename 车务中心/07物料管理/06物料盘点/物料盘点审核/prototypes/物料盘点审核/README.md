# 物料盘点审核 原型

## 页面列表
- index.html - 物料盘点（审核版）完整原型

## 使用说明
1. 直接用浏览器打开 index.html
2. 支持 URL 参数快速跳转到指定视图：
   - `?view=edit` - 编辑/新增盘点单页面
   - `?view=detail` - 盘点明细Tab
   - `?view=drawer&no=PD202603001&status=approved` - 打开指定盘点单详情抽屉

## 交互说明

### 盘点单列表
- 状态筛选支持：草稿、待审批、审批中、已通过、已驳回、已撤销
- 各状态对应不同操作按钮：
  - 待审批 → 取消审批
  - 审批中 → 无操作（仅查看）
  - 已通过 → 撤销
  - 已驳回 → 修改 / 删除
  - 草稿/已撤销 → 编辑 / 删除
- 点击盘点单号打开详情抽屉

### 详情抽屉
- 基本信息 + 审批信息卡片（含审批进度可视化）+ 盘点汇总 + 盘点明细
- 审批进度以节点形式展示（已完成-绿色、当前-蓝色、待处理-灰色）
- 已驳回状态展示驳回原因（红色高亮）
- 底部操作按钮根据状态动态显示

### 弹窗
- 提交确认弹窗：含审批流程提示文案
- 取消审批确认弹窗
- 驳回后修改保存确认弹窗
- 撤销确认弹窗
