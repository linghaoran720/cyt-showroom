# 2605 维修保养导入优化 · 交互原型

## 打开方式

用浏览器直接打开：

`index.html`

（双击或通过「文件 → 打开」；无需本地服务器。）

## 原型范围（对齐 PRD v1.8）

- 列表页占位 + **「导入」** 打开右侧抽屉。
- **阶段一（前端）**：三项必填、数量/单价等**有值时**格式校验；**校验结果**列仅反映前端。
- **提交**：模拟服务端逻辑（同组车牌/类型一致、维修编号是否与列表已存在重复）；**Toast** 成功或报错。
- 演示按钮：示例数据、**同编号车牌不一致**（前端通过、提交失败）。
- 上传前覆盖确认；提交成功后中间表**只读**示意。
