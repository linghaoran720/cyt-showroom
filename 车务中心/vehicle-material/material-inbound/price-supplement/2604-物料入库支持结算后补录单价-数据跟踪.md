- 事件：`material_price_fill_click`、`material_price_fill_submit`、`material_price_fill_success`
- 数据平台：**[待补充数据平台链接]**

**数据库查询**：
```sql
-- 功能使用客户数
SELECT COUNT(DISTINCT company_id)
FROM MaterialInboundBill
WHERE is_price_filled = 1
AND price_fill_time >= '2026-XX-XX';

-- 补录价格操作次数
SELECT COUNT(*)
FROM MaterialPriceFillLog
WHERE fill_time >= '2026-XX-XX';

-- 单价为空的入库单占比
SELECT
  COUNT(CASE WHEN buy_unit_price IS NULL THEN 1 END) * 100.0 / COUNT(*) as empty_price_rate
FROM MaterialInboundBill
WHERE created_at >= '2026-XX-XX';
```

## 📊 使用说明

### 如何使用本文档

1. **设定目标**：在功能上线前，补充所有"**[待补充]**"的目标值
2. **收集数据**：功能上线后，定期（每周/每月）从数据平台获取实际数据
3. **补充数据**：将实际数据填入对应的表格中
4. **计算达成率**：达成率 = 实际值 / 目标值 × 100%
5. **评估表现**：根据达成率评估表现（优秀/良好/及格/不及格）
6. **分析原因**：分析数据表现的原因（为什么好/为什么差）
7. **制定优化**：根据分析结果制定优化建议和行动计划
8. **持续跟进**：定期更新数据，持续跟进优化效果

### 评估标准

- **优秀**：达成率 ≥ 120%（超过挑战目标）
- **良好**：达成率 100-119%（达到期望目标）
- **及格**：达成率 80-99%（达到基础目标）
- **不及格**：达成率 < 80%（未达到基础目标）
